# SEO AI Suite V1

All-in-one SEO AI Suite starter platform.

Modules: Dashboard, Rank Tracker, Technical Audit, AI SERP Checker, Keyword Clustering, Content Briefs, Competitor Analyzer, Reports.

Use the same Render settings as before:

Build Command: `pip install -r requirements.txt`

Start Command: `gunicorn app:app`

Environment Variable: `SERPER_API_KEY`
